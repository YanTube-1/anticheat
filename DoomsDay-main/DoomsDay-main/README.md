## 📥 Installation

1. Make sure you have **Java** installed on your system.  
2. Download the latest mod file here:  
   [Releases Page](https://github.com/patlozer/DoomsDay/releases)
3. Follow instructions for your loader:  
   - **Minecraft Launcher / MultiMC / Prism Launcher / TLauncher**: Drop the mod `.jar` into the `.minecraft/mods` folder, make sure you have **Forge** installed.
   - **Forge / LabyMod**: Use the appropriate loader to add the mod  
4. Launch Minecraft with your preferred loader and enjoy!
